# AntBox Operations

## Daily Health Checks

- tuner availability and signal quality
- daemon process health
- upload latency and reconnect counts
- local disk headroom for buffers

## Common Incidents

- tuner lock failure
- weak/no signal
- hardware throttling due to temperature
- network instability to antserver

## Operational Requirements

- remote restart capability
- persistent logs for debugging
- clear reprovisioning workflow for replacement hardware
