# AntServer Operations

## Key Metrics

- ingest uptime
- live event success rate
- schedule execution lag
- recording finalization latency

## Common Operational Risks

- schedule drift due to metadata source errors
- ingest overload during concurrent events
- delayed post-processing due to queue saturation

## Required Runbooks

- live event fails to start
- ingest drops mid-event
- recording not published to TV catalog
